「 Project Goals 」

上面有D引言, 落下小小, 先講Project Goals

第1點, 重組內容策略，增加用戶探索容易性 同內容質量，增强 User參與度, 同User停留時間，從而減低Bounding rate

第2點, 重新評估 在線形象和體驗，同品牌保持一致

跟住第二個 Goal, 同 UMG Hong Kong 聯盟行銷既合作關係，可以推動更多銷售機會。 下面是角色框架：

我地提供 Platform 即系Website, 佢地提供 Product 即系巨地CD

開會時已好快揾到問題, Bounce rate太高, 連帶到 Ushop消費者顯著下降, 跟住同 UMG marketing team一齊做 online Survey, 有28位, 整理左之後, 同teammate評估後, 發現到有3個 Pain point:

有43%人表示, 唔太似一個音樂網站, infomation 有D雜亂, CD專輯過多Home page, Content結構唔太好, 還提到Search result唔正確唔清析, 影響購物的意圖。

有55%人表示, 負面體驗增加左 Bounding rate, 冇 Responsive website 令user沮喪, 同埋個 Design out date左。

有39%人表示, 冇提供 Social content, 他們想睇有咩 Artist的 Facebook live 或者 Event。 不過，舊 Website 只顯示新聞和CD。 同目標受眾有距離。

之後做左 Competitive Analysis, 去理解市場上 Music brand Platforms。

「 IDEATION & SOLUTION 構思與解決方案 」

「 Problem Statement 」

我們如何重新設計 UMG 數字在線體驗，降低跳出率，以及通過新平台提高參與度並推動更多銷售？”

同 同事 Brainstorm之後, 綜合左 3個Pain point, 提出左3個構想

#1 專注內容可發現性, 清晰直觀內容架構, 對大Brand糸好重要, 首先從IA, Site map上着手, 減少過多Subpage, 避免不必要的點擊, 另外, Landing page 內容都要重新整理過, 排序, 表達方法, 同埋減少不需要的內容。

另外, 改進Search果方面, Result會詳細清晰D, 如 Search CD 會有CD cover image, 精準D, 揾到User所需。

#2 增強視覺增加吸引力, 尤其係音樂品牌係重要, 可以提高識別率, Stand out brand image, 要 Trendy同時要 keep內容可讀性。

#3 減低 Brand與 Target audience之間的差距, 首先 Responsive design, 可以糸唔同 Devices睇到, 目的係, 可以快速有效地並降低跳出率, 另外, 減少過多 CD專輯類似內容, 提供番相應 Social content比 User, Facebook live, Soical feed等等, User可以停留更長時間, 推動更多的流量和銷售。

1. 以轉化為中心的Landing

主要解決番第1個 User pain point, 唔太似一個音樂網站, Infomation有D雜亂, Content結構唔太好 既問題

所以我地要增强番內容清晰直觀, Layout層次感, 同埋內容可發現性。

重新Design後, 整個Landing 只有三個內容部分, 主要目的係, 令 User 易於發現, 同埋相關既後續旅程。

第1部分, 好似1個 大字報 layout, 有10 grid 不同sytle, 有tag指示番相應既內容, 例如 Highlight CD, 重要event, Hot news等等, 之後, 專輯詳情 page, 有個圖形CTA button, click 去Ushop payment system 進行購買。

第2部分, 有個音樂大師特集 Slider

第3部分, 有個試聽 Popup audio player widget

之後, 專輯詳情 page, 有個圖形 CTA button, 去 Ushop payment system 進行購買。

另外, 重新 Design search, 係 Full page layout, 目的係, 唔需要跳去Result page, 入左 Key word後, 有查詢建議 同埋 CD thumbnail, 可以容易揾到巨地想要。

2. 增强番視覺吸引力, 係針對第2 Pain point, 唔太似一個音樂網站, 過時的設計 既問題

整個 Website 係 Responsive, 可以糸唔同 Devices睇到, 主要係解決 brounding rate, User 嚟到website fit 番巨地既screen。

用左深灰底色 Design 感覺 Trendy D, 可以突出D image, 例如CD Cover, Artist images, Poster art, 另外, 係Design system 做左草書element 識別唔同類型CD。

仲有D User interaction, 例如試聽 Popup audio player, mouse over歌名會有個CD cover, 同埋係UMG PRESS page, 有個下拉menu, click 去3個sub section。

3. 加左Social content, 減低 Target users之間差距, 係解決第3 Pain point 既問題

係注目動態 page, users可以睇到upcoming event, 可以click "Join" button, 有個Dialogue 可以syn去 自己既 Google and Apple Calendar, Keep track 到自己有興趣event, 唔洗miss左。

係藝人社交 page, Embed左 Artist facebook feed 係裡面, 可以吸引留意 Artist soical media user, 同埋Keep users 停留長D時間, 另外好處係, Keep住有新content 而唔需要發布新內容。
